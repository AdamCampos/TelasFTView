Clicando no botão Submeter, uma página JSP será chamada.
Ao ser aberta, esta página irá trazer uma caixa de escolha de arquivos.
Nesta caixa de diálogo só é possível escolher pastas.
Deve-se ser selecionada uma pasata onde há arquivos XML para sofrerem o parse.
Todos os arquivos irão para a memória, pois o parser utilizado será o DOM.
Desta forma será possêvel se navegar pela árvore de Nodes.
